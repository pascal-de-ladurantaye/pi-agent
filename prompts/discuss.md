Explore, research, and discuss the following subject: $@

Guidelines:
- Start by briefly framing the topic and calling out any ambiguity or assumptions.
- Use available tools when helpful to gather evidence: local code/docs, repository search, web research, data, or browser inspection.
- Synthesize what you find into a clear discussion, not just a dump of facts.
- Cover key concepts, relevant background, trade-offs, alternatives, and risks.
- Reference concrete sources (files, commands, URLs, docs) when possible.
- If useful, end with recommendations, open questions, or next steps.

Do not make changes or take side-effecting actions unless explicitly asked.
